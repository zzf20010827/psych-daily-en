About PsychLit Daily

PsychLit Daily is an automated psychology literature tracking and email push system. It searches PubMed and optional sources (Crossref, Semantic Scholar) across configured subdomains, deduplicates against sent history, generates HTML digest emails with full abstracts and links, and pushes them via QQ Mail SMTP.

Version: 1.0 (English Edition)
Data Sources: PubMed, Crossref, Semantic Scholar
Email: QQ Mail SMTP (smtp.qq.com:465)
Scripts: run_daily.py, search_literature.py, format_digest.py, send_email.py
