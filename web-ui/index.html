About PsychLit Daily

PsychLit Daily is an automated psychology literature tracking system. It searches PubMed and optional sources (Crossref, Semantic Scholar) across configured subdomains, deduplicates against published history, generates HTML digests with full abstracts and links, and publishes them to the public homepage.

Version: 1.0 (English Edition)
Data Sources: PubMed, Crossref, Semantic Scholar
Scripts: run_daily.py, search_literature.py, format_digest.py
